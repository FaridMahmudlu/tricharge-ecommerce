# 3-in-1 Wireless Charging Station E-commerce Website

This is an e-commerce website built with React, TypeScript, and Styled Components to showcase and sell the 3-in-1 Wireless Charging Station.

## Features

- Modern, responsive design
- Product showcase on homepage
- Detailed product pages with specifications
- Shopping cart functionality
- Multiple product views and variants
- Bulk order pricing information

## Product Information

The website is designed around selling the 3-in-1 Wireless Charging Station with the following specifications:

- 15W/10W/7.5W/5W charging capability
- Charges phone, smartwatch, and earphones simultaneously
- Made with ABS and Aluminum
- Short circuit, overcurrent, and overvoltage protection
- 1-year warranty
- Input: 9V/3A
- Output: 9V/1.5A

## Getting Started

1. Install dependencies:
```
npm install
```

2. Start the development server:
```
npm start
```

3. Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

## Adding Real Product Images

See the [PRODUCT_IMAGES_INSTRUCTIONS.md](./PRODUCT_IMAGES_INSTRUCTIONS.md) file for detailed instructions on how to add real product images to the website.

## Customization

You can easily customize:

- Product details in `src/data/products.ts`
- Homepage content in `src/pages/Home.tsx`
- Styling through the styled-components in each file

## Building for Production

```
npm run build
```

This builds the app for production to the `build` folder. It correctly bundles React in production mode and optimizes the build for the best performance.
